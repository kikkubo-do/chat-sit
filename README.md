# Chat SIT

This repository contains the source code and documentation for the Chat SIT project.

## Overview
Chat SIT is designed to facilitate seamless communication through an intuitive chat interface. 

## Features
- Real-time messaging.
- User authentication.
- Group chats.
- Support for various media types.

## Getting Started
To get a local copy up and running follow these simple steps.

### Prerequisites
- Node.js
- MongoDB

### Installation
1. Clone the repo: `git clone https://github.com/kikkubo-do/chat-sit.git`
2. Install NPM packages: `npm install`
3. Go to `.env` and set your credentials.
4. Start the application: `npm start`
